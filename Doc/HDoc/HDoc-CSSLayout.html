Episode 1 | Display

div memiliki default display:block dan span memiliki default display:inline. semua elemen hanya memiliki 2 default display, block atau inline.

4 value dari display

  • inline

    • Tidak menambah baris baru, lebar dan tinggi hanya sebesar konten didalamnya, tidak dapat mengatur tinggi dan lebar pada inline, margin dan padding hanya horizontal tidak vertikal

  • inline-block

    • Harus diset secara manual, perilaku dasarnya seperti inline, perbedaannya inline-block dapat mengatur tinggi dan lebarnya

  • block

    • Secara default menambahkan baris baru, defaultnya akan memenuhi lebar browser/parent-nya, dapat diatur tinggi dan lebarnya, block dapat menyimpan inline, inline-block atau block lagi

  • none

    • digunakan untuk menghilangkan elemen

Episode 3 | Box Model

Margin, Padding, Border, dan Content. Pada box model, yang dapat diatur hanya margin, padding, dan border.

property setiap margin terdiri dari, margin-left, margin-right, margin-top, margin-botom, dan margin untuk secara keseluruhan sisinya. padding dan border memiliki semua property yang tadi

Episode 4 | Box Model: Margin

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
  • margin

Overlapping Margin: Terjadi ketika menggabungkan 2 margin (misal, margin-left dan margin-right) antar 2 elemen

Overlapping Vertikal: akan mengambil nilai yang lebih besar tanpa menambahkan. misal margin-botom 150px dari elemen A dan margin-top 50px dari elemen B. maka, tidak ada perubahan atau penambahan jadi 200px antar elemen A dan B.

Overlapping Horizontal: akan menambahkan margin-right dan margin-left dari 2 elemen yang saling overlap

Negatif Margin: memberikan nilai negatif -0px untuk menggerakan ke arah sebaliknya. misal margin-left: -150px, maka daripada mengeser ke arah kanan 150px, maka yang terjadi adalah menggeser ke arah kiri sebesar 150px.

Auto Margin: hanya pada margin-left dan margin-right. hal ini akan menempatkan ke tengah-tengah dari parentnya.

Episode 8 | Float

Float merupakan properti CSS untuk mengatur posisi elemen. Float dapat memaksa elemen untuk berada di sebelah kiri atau kanan dari parent/pembungkusnya. Hal ini menjadikan elemen yang memiliki property float akan dianggap tidak ada/menjadi display inline

value float: none (default), left, right

Normal flow (display) vs Out-of flow (float)

Jika semua konten dari suatu pembungkung menjadi float, maka nanti pembungkus/kontainernya akan collapse

Episode 9 | Clear

Mengatasi masalah container/pembungkus collapse

3 cara: menggunakan overflow:auto, div kosong, teknik micro clearfix

overflow: auto, digunakan pada parent dari elemen yang menggunakan float

Property css clear: left, right, both. digunakan untuk menghentikan/membersihkan float.

Cara menggunakannya adalah dengan membuat div kosong pada akhir elemen yang ingin menggunakan float, lalu tambahkan style clear:left,right, atau both pada divnya

Micro clearfix: menggunakan script yang ada di google

Episode 11 | Position: Static & Relative

Digunakan untuk memindahkan/menggeser elemen tanpa mempengaruhi elemen disekitarnya. tidak seperti margin

css property position: static (default), relative, absolute, fixed

Elemen yang diberi position selain value static, maka dapat menggunakan property top, left, bottom, dan right untuk mengatur posisinya

Position:relative; posisi relative ini bergerak mengacu pada posisi awalnya, posisi ini juga tidak menghilangkan posisi awalnya di "dimensi" sebelumnya, top:0; dan left: 0; tidak mengubah posisi ini.

position:relative; biasanya hanya digunakan untuk mengubah "dimensi" dari suatu elemen. position:absolute; yang biasanya digunakan untuk menggerakan posisi-posisi menggunakan top, right, left, bottom.

Episode 12 | Position: Absolute & Fixed

Sama seperti relative, absolute juga memindahkan "dimensi" elemennya. hanya saja, jika pada relative posisinya masih ada dan menjadi acuan posisi relative. pada absolute, posisinya akan diisi oleh elemen selanjutnya.

position absolute bergerak berdasarkan posisi dari elemen parentnya yang berada pada dimensi (non-static) yang sama dengan elemen absolute.

position fixed selalu bergerak berdasarkan windownya (body) walaupun ada non-static terdekatnya

Episode 13 | Z-Index dan Penggunaan lain dari properti position

Z-Index digunakan untuk mengatur tumpukan dari elemen non-static (absolute, fixed, relative). z-index:; | semakin tinggi nilai z-index maka elemennya akan makin terdepan (secara tumpukan)
